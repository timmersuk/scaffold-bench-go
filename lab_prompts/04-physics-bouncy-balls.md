---
title: Bouncy Balls Physics
category: physics-sim
---

Write a single self-contained HTML file (no external dependencies, no frameworks, no CDN) that implements a 2D physics simulation of bouncy balls. Requirements:

- Pure vanilla JS + HTML + CSS only, inline in one file.
- Use an HTML5 `<canvas>` sized to fill the viewport.
- Balls have position, velocity, radius, and mass.
- Implement gravity, elastic collisions between balls, and wall bouncing.
- Click/drag to launch new balls into the simulation.
- At least 20 balls active simultaneously, colorful, with size variation.
- Trail effect: semi-transparent background clear for motion blur.
- Smooth 60fps animation.
- Show a small HUD with ball count and FPS.
- Physics should feel natural — balls should bounce off each other realistically.
