---
title: Haiku Trio
category: creative-writing
---

Write exactly three original haikus (5-7-5 syllable structure each) as a single HTML page styled with beautiful typography. Requirements:

- Pure HTML + CSS only, inline in one file, no external dependencies or CDN.
- Three distinct haiku themes: one about nature, one about technology, one about human emotion.
- Elegant, minimalist design with lots of whitespace.
- Each haiku on its own section with a subtle decorative line or icon between them.
- Use system fonts styled beautifully (no Google Fonts).
- A dark theme with a warm accent color.
- A small footer with "— written by an AI" in muted text.
- The page should feel like a zen garden for reading poetry.
